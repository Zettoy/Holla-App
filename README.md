## Holla Readme
This project consists of an Android front end (contained in this repo),
and a REST API backend contained in another repo.

## Installation
- Download & install Android Studio [here](https://developer.android.com/studio/).
- Clone this repo to your computer.
- Get a Google Maps API key [here](https://developers.google.com/maps/documentation/android-sdk/signup).
- Save the Google Maps API key to `/app/src/main/res/values/google_maps_api.xml`.
- In Android Studio, open the project folder.
- Setup an Android Virtual Device.
- Run/debug the app, following instructions [here](https://developer.android.com/studio/debug/).
